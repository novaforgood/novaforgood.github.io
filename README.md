# Nova's Website

https://www.novaforgood.org/

Built using Gatsby.js and Contentful :)

## Develop

Add Contentful keys to `.env.development` file:
```
CONTENTFUL_SPACE_ID=
CONTENTFUL_ACCESS_TOKEN=
```

Then, to start the project:
```
yarn
yarn start
```

## Deploy

Add Contentful keys to `.env.development` file:
```
CONTENTFUL_SPACE_ID=
CONTENTFUL_ACCESS_TOKEN=
```

Then, to deploy:
```
yarn deploy
```
